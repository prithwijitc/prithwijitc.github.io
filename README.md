# prithwijitc.github.io

Personal academic website for Prithwijit Chowdhury, built with the [al-folio](https://github.com/alshedivat/al-folio) Jekyll template.

## Local preview

Requires Ruby 3.3.x, matching the GitHub Actions deploy workflow.

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.

For Docker-based setup and deployment notes, see `INSTALL.md`.
